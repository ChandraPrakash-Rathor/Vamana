# Vamana - Premium Perfume E-Commerce Website

A modern, fully-featured e-commerce website for luxury perfumes and attars built with React.js. Features a beautiful UI with sand-themed design, comprehensive shopping experience, and festival-specific sale animations.

## 🌟 Features

### Core Features
- **Responsive Design**: Fully responsive across all devices (mobile, tablet, desktop)
- **Modern UI/UX**: Sand-themed color palette with smooth animations
- **Product Catalog**: Browse perfumes and attars with filtering options
- **Shopping Cart**: Add to cart, update quantities, apply coupons
- **Secure Checkout**: Multi-step checkout with payment options (Card, UPI, COD)
- **Order Tracking**: Real-time order tracking with timeline
- **Invoice Generation**: Professional PDF-ready invoices

### Pages
1. **Home Page** - Hero slider, Signature Collection, Limited Offers, Best Sellers, Testimonials
2. **Catalog Page** - Filter by category (All/Perfumes/Attars/Combo)
3. **Product Detail Page** - Image gallery, size selection, fragrance notes
4. **Shopping Cart** - Item management, coupon codes, order summary
5. **Checkout Page** - Personal info, shipping address, payment methods
6. **Sale Page** 🎉 - Festival-specific animations (Diwali, Christmas, New Year, etc.)
7. **Track Order** - Order tracking with visual timeline
8. **Returns & Shipping Info** - Policy details and shipping zones
9. **Invoice Page** - Professional tax invoice with GST

### Special Features
- **Splash Screen**: Animated video splash on page load
- **Quick View Modal**: Preview products without leaving the page
- **Breadcrumb Navigation**: Easy navigation tracking
- **Scroll to Top**: Floating button for easy navigation
- **Auto Scroll on Route Change**: Pages start at top on navigation
- **Festival Animations**: Dynamic animations for 10+ festivals

## 🛠️ Tech Stack

- React.js 18
- React Router DOM v6
- Bootstrap 5
- Font Awesome
- Custom CSS with Sand Theme

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/ChandraPrakash-Rathor/Vamana.git
cd Vamana

# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

## 🎨 Design System

### Color Palette (Sand Theme)
- Sand shades from 100 (lightest) to 950 (darkest)
- Primary: #c8a45d
- Accent: #b8935a

### Typography
- Headings: Playfair Display
- Body: Roboto
- Responsive with clamp()
